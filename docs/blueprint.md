# **App Name**: StreamCart Login

## Core Features:

- Splash Screen: Splash screen with StreamCart logo.
- Login Form: Login page with email/phone and password inputs.
- Forgot Password Link: Remember password
- Login Button: Login button with arrow.
- Signup Link: Link to signup page for new users.

## Style Guidelines:

- Primary color: The StreamCart red (#FF0000), to convey urgency and encourage users to purchase quickly.
- Background color: Very light grey (#F0F0F0), for a clean background that won't distract.
- Accent color: A slightly darker grey (#B0B0B0) is applied for text input backgrounds and other secondary information.
- Font: 'Inter' (sans-serif), is used to keep the interface modern, clear, and machine-like.
- Split the screen into two sections: one for the splash screen and one for the login form.
